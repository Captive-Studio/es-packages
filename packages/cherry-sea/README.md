# Cherry Sea _(@captive/cherry-sea)_

> Theme for Tailwind or vanilla CSS

## Installation

NPM :

```sh
npm install @captive/cherry-sea --save
```

Yarn :

```sh
yarn add @captive/cherry-sea
```

## Usage

### Javascript import

```javascript
import { colors } from '@captive/cherry-sea/dist/colors.js';
```

### Tailwind

```javascript
import { colors } from '@captive/cherry-sea/dist/colors.js';

module.exports = {
  theme: {
    extend: {
      colors: colors
    }
  }
}
```
